# TD_learning_pacman
Teaching pacman how to escape its ghost enemies through temporal difference learning.
Commands to run different versions:   
`python pacman.py -p ApproximateQAgent -a extractor=SimpleExtractor -n *FPS* -l *GRID_LAYOUT*`  
`python gridworld.py -a q -k 50 -n 0 -g BridgeGrid -e *NUM_HOLES*`
